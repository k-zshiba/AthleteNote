# AthleteNote

運動している人を対象としたWebアプリケーションです。
体調や体重などの数値や練習内容を記録し、可視化することで競技力向上のためのツールとして使用することが出来ます。
Twitterと連携して登録した練習内容を投稿することが出来ます。


# URL
https://athlete-note.herokuapp.com/
ゲストログインを押してもらうと登録せずに利用できます。
